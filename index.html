Excel • Valuation

3-Statement Financial Model & DCF Valuation – Apple Inc.

A fully integrated financial model built using Apple’s 2024 financial statements. The model forecasts operating performance, constructs free cash flow (FCF), and estimates intrinsic value through discounted cash flow (DCF) valuation with dynamic scenario analysis.

DCF Valuation Free Cash Flow NOPAT WACC Terminal Value Scenario Analysis
Model Structure Income statement, operating balance sheet, FCF model, valuation bridge, and scenario-driven outputs.
Valuation Approach Two-stage DCF model with Gordon Growth terminal value and enterprise-to-equity value conversion.
Excel Skills XLOOKUP-driven scenario selection, dynamic inputs, financial modeling, and structured valuation logic.
Key Output Intrinsic value of ~$184/share with sensitivity to WACC and terminal growth assumptions.

Interactive Financial Performance Dashboard – Apple vs. Microsoft

An interactive Excel dashboard built to analyze and compare Apple and Microsoft across key financial metrics, including revenue, net margin, ROE, free cash flow, debt ratio, and asset turnover. The model includes dynamic company and year selection for KPI retrieval while the charts continue to display full historical trends.
